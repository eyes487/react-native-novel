# react-native-novel
一个使用react-native爬取小说观看的APP

rn版本 0.60.5

# 运行方法
1. 克隆代码：

```
git clone https://github.com/eyes487/react-native-novel.git
```

2. 进入项目根目录并安装依赖：

```
cd react-native-novel
npm install or yarn
```

3. 运行项目
```
react-native run-android

```
